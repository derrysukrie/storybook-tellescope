# Form Components

A collection of form components designed to create consistent and accessible form layouts.

## Overview

The Form components consist of three main parts:

- **FormGroup**: A wrapper component that provides structure to form elements
- **FormGroupLabel**: A label component with customizable size and state
- **FormHelperText**: A helper text component for providing additional information or error messages

These components are built on top of Material UI's form components but with a standardized API and styling.

## Usage

```tsx
import { FormGroup } from './FormGroup';
import { FormGroupLabel } from './FormGroupLabel';
import { FormHelperText } from './FormHelperText';
import { TextField } from '@mui/material';

<FormGroup>
  <FormGroupLabel>Email Address</FormGroupLabel>
  <TextField 
    hiddenLabel 
    variant="filled" 
    placeholder="Enter your email..." 
  />
  <FormHelperText>
    We will never share your email with anyone.
  </FormHelperText>
</FormGroup>
```

## Component API

### FormGroup

A container component that wraps form elements.

```tsx
<FormGroup>
  {/* Form elements go here */}
</FormGroup>
```

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| children | React.ReactNode | - | The content of the form group |

### FormGroupLabel

A label component for form groups.

```tsx
<FormGroupLabel labelSize="default">Label Text</FormGroupLabel>
```

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| children | React.ReactNode | - | The label text |
| labelSize | 'default' \| 'large' | 'default' | Controls the size of the label |
| error | boolean | false | Whether to display the label in error state |
| disabled | boolean | false | Whether to display the label in disabled state |

### FormHelperText

A component for displaying helper text or error messages.

```tsx
<FormHelperText error={false}>Helper text goes here</FormHelperText>
```

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| children | React.ReactNode | - | The helper text content |
| error | boolean | false | Whether to display the text in error state |
