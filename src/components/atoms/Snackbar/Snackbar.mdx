# Snackbar

The Snackbar component provides a way to display brief messages to users in a non-intrusive manner.

## Basic Usage

```tsx
<Snackbar
  open={true}
  message="This is a snackbar message"
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| open | boolean | - | Controls whether the snackbar is displayed |
| message | string | "This is a snackbar message" | The text content of the snackbar |
| asAlert | boolean | false | When true, displays the snackbar with alert styling (red background) |
| action | React.ReactNode | - | Optional action elements to display in the snackbar |

## Examples

### Default Snackbar

```tsx
<Snackbar
  open={true}
  message="This is a default snackbar message"
/>
```

### Alert Snackbar

```tsx
<Snackbar
  open={true}
  message="Warning! Please check your input"
  asAlert={true}
/>
```

### With Action Buttons

```tsx
<Snackbar
  open={true}
  message="Message sent successfully"
  action={
    <>
      <Button 
        size="small"
        variant="text"
        color="inherit"
        onClick={handleClose}
      >
        UNDO
      </Button>
      <IconButton
        size="small"
        aria-label="close"
        color="inherit"
        onClick={handleClose}
      >
        <CloseIcon fontSize="small" />
      </IconButton>
    </>
  }
/>
```

## Styling

The component uses Material UI's Snackbar and Alert components with custom styling:
- Default: Dark blue background (#151B2C)
- Alert: Red background (#BA1A1A)
- Enhanced box shadow for better visibility
