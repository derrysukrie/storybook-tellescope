# FormBuilder

A dynamic form builder component with three distinct color schemes. Features interactive states with a checkcirlce icon that changes color based on selection status and modular header/footer structure.

## Features

- **Three Color Schemes**: Choose from tellescope-soft, tellescope, or agua themes
- **Selected State Control**: Toggle between selected (green checkcirlce) and unselected (grey checkcirlce) states

## Color Schemes

### Tellescope Soft (Purple-Blue Theme)
- **Primary**: #798ED0 (Blue-Purple)
- **Secondary**: #585E72 (Dark Grey)
- **Accent**: #DDE1F9 (Light Blue-Purple)
- **Background**: #F4F3FA (Light Purple-Grey)

### Tellescope (Blue Theme)
- **Primary**: #4A90E2 (Blue)
- **Secondary**: #2C3E50 (Dark Blue-Grey)
- **Accent**: #A8D8F0 (Light Blue)
- **Background**: #F8FBFF (Very Light Blue)

### Agua (Aqua/Teal Theme)
- **Primary**: #00B4D8 (Cyan)
- **Secondary**: #0077B6 (Dark Blue)
- **Accent**: #90E0EF (Light Cyan)
- **Background**: #F0FDFF (Very Light Cyan)

## Usage

### Basic Usage

```jsx
import { FormBuilder } from './FormBuilder';

function App() {
  return (
    <FormBuilder type="tellescope-soft" />
  );
}
```

### Different Types

```jsx
import { FormBuilder } from './FormBuilder';

function App() {
  return (
    <div>
      <FormBuilder type="tellescope-soft" selected={false} />
      <FormBuilder type="tellescope" selected={false} />
      <FormBuilder type="agua" selected={false} />
    </div>
  );
}
```

### Selected State

```jsx
import { FormBuilder } from './FormBuilder';

function App() {
  return (
    <FormBuilder type="agua" selected={true} />
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `"tellescope-soft" \| "tellescope" \| "agua"` | **Required** | The type of form builder with different color schemes |
| `selected` | `boolean` | `false` | Whether the component is in selected state. Controls the checkcirlce icon color. |

## Component Structure

The FormBuilder component is composed of three main parts:

### FormWrapper
- Main container with border, background, and hover effects
- Handles the overall styling and interactive states
- Background color changes based on type

### FormHeader  
- Contains the form categories (Primary, Secondary, Accent)
- Houses the CheckCircle icon with conditional coloring
- Text colors adapt to the selected type theme

### FormFooter
- Bottom section with white background
- Contains branding text that matches the type ("Tellescope Soft", "Tellescope", "Agua")
- Separated from header with subtle border
